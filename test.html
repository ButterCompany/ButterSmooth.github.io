Creating a new paragraph is as easy as a pea. We can just use the (p) tags.
Making the new sentence into a new line uses the (br) tags.

(br),(img) and (hr) tags are empty tags. We do not need to close them.

Targets can be used in html hyperlink.
(_blank) leads to the link opening to a new tab.
(_self) leads to the link opening inside the existing tab.
